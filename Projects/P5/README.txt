twisted.c
-----------------------------------------------------------------------------------------------
This program represents a game of twisted. It is based of hangman. However, the computer changes
the word mid game based on the pattern of the word to make the game harder.
-----------------------------------------------------------------------------------------------

Compile command:
        g++ -g -Wall twisted.cpp twistedf.cpp -o twisted

Or use provided Makefile:
   make clean
   make

Run command:
    ./twisted [textfile.txt] or ./twisted

(Commands must be executed within the project directory)
