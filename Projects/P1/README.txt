craps.c
-----------------------------------------------------------------------------------
This program prompts the user for a balance for a game of craps. Then, the
user is prompted to enter a wager for the game. Once the wager is entered,
a game of craps is ran. Then, either the player is then prompted if they want to play
another game or the game ends if balance is 0.
