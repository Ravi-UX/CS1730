yosh.c
-----------------------------------------------------------------------------------------------
This program is a basic shell which runs any Linux command and a few built in commands.
-----------------------------------------------------------------------------------------------

CODER: Ravi Cheekati
CREDIT/CONTRIBUTIONS:



Compile command:
        gcc -g -Wall yosh.o parse.o list.o -o yosh

Or use provided Makefile:
   make clean
   make

Run command:
    ./yosh

(Commands must be executed within the project directory)
