sfind.c
-----------------------------------------------------------------------------------------------
This program takes a starting directory, a substring, and a file type and prints out any files
which contains the substring in its name and is of the same file type.
-----------------------------------------------------------------------------------------------

CODER: Ravi Cheekati
CREDIT/CONTRIBUTIONS:
Lab 07


Compile command:
        gcc -g -Wall sfind.c sfindFunctions.c -o sfind

Or use provided Makefile:
   make clean
   make

Run command:
    ./sfind -s [dirname] [-n name] [-t type]

(Commands must be executed within the project directory)
